# Percentage & Grade Finder 📊

A simple web application that helps students quickly calculate their percentage and grade based on marks.

## 🚀 Features
- Calculate percentage from obtained & total marks
- Find grade based on percentage
- Input validation for accurate results
- Multi-page navigation (Home, Percentage, Grade)
- Responsive design with modern UI

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 📂 Project Structure
- index.html → Home page
- percentage.html → Percentage calculator
- grade.html → Grade calculator
- style.css → Styling
- script.js → Logic

## 📚 What I Learned
- Form handling & validation in JavaScript
- Multi-page website structure
- Responsive design & UI improvements
- Implementing calculation logic

## ▶️ How to Run
Open `index.html` in your browser and navigate through pages.
